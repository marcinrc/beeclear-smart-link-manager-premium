== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **BeeClear Smart Link Manager**.
3. Install and activate the BeeClear Smart Link Manager plugin.

= Manual installation =

1. Upload the entire `beeclear-smart-link-manager` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the **BeeClear Smart Link Manager**.

== Frequently Asked Questions ==

= Does this plugin work with the latest WordPress? =
Yes. The plugin is tested up to WordPress 6.9.

= Does it modify existing content? =
No. Links are managed dynamically without permanently changing post content.

= Can I limit the number of links per page? =
Yes. You can set per-page link caps and custom priorities for competing targets.

= How do I rebuild the index after changing rules? =
Go to the plugin settings and use the manual rebuild button, or enable automatic rebuilding on post updates.

= Why are some phrases not linking? =
Check rule priority, page-level link limits, and whether case-sensitivity or phrase matching settings exclude the text.

= Can I exclude specific posts or post types from auto-linking? =
Yes. Use the plugin settings to disable linking for selected post types or individual posts.

= How can I migrate rules between sites? =
Use the import/export tools to move rules between environments without re-creating them manually.
